What is GENEXUS UI?

GENEXUS UI is a demo project built to showcase a modern, premium AI image generation interface. It is designed as an example of what a professional-grade web application for AI art generation could look like.

This project demonstrates clean UI/UX design, responsive layouts, theme switching, local history management, and integration with a simple image generation API.

Key Features

  • Flux-Schnell Engine — Powered by a fast image generation model.
  • Responsive Design — Works seamlessly on desktop, tablet, and mobile.
  • Dark & Light Themes — Switch between themes with persistent preferences.
  • Local History — Your recent generations are saved in your browser.
  • Customizable Settings — Aspect ratio, quality, style, and count controls.
  • One-Click Actions — Download images or copy prompts instantly.

Tech Stack

  • Vanilla HTML5, CSS3, and JavaScript (no frameworks)
  • Lucide Icons for a consistent icon set
  • Google Fonts (Plus Jakarta Sans)
  • LocalStorage for theme and history persistence
